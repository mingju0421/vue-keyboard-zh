# vue-keyboard-zh

```JavaScript
<Keyboard v-model="value" layout="normal"></Keyboard>
```

| 参数    | 说明                                             | 类型   | 可选值         | 默认值 |
| :------ | :----------------------------------------------- | :----- | :------------- | :----- |
| v-model | 绑定值                                           | string | -              | -      |
| layout  | 键盘类型,normal: 普通字母键盘, numeric: 数字键盘 | string | normal/numeric | normal |
